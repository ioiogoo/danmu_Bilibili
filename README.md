# danmu_Bilibili
哔哩哔哩弹幕转化器
# How to use it
下载bilibili视频上的弹幕并转换为ass字幕，可在支持ass字幕播放器上使用

执行`Bilizimu('av2271112').Genera_ass()`即可在当前目录生成ass字幕
